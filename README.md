# tonmoy-TS.github.io

Personal portfolio, published via GitHub Pages. Plain HTML/CSS/JS, no build step.

## Structure

- `index.html` — single-page site (About, Education, Experience, Skills, Projects, Publications, Contact)
- `css/style.css` — styles
- `js/main.js` — mobile nav toggle + active-section highlighting on scroll
- `assets/img/` — images (add a real photo here and reference it in the hero section)

## Local preview

Open `index.html` directly in a browser, or serve it locally, e.g.:

```
npx serve .
```

## Deploy

Push to `main`. GitHub Pages serves the repo root directly (see repo Settings → Pages).
